These scripts are serial implmentations of ADMM for various regularizations.
Concluding Logistic with various regulars (such as Lasso,Lhalf,SCAD and MCP) by ADMM algorithm and distributed Logistic with various regulars by ADMM algorithm.
The coding is according to the https://web.stanford.edu/~boyd/papers/admm/.
The file Reference has papers about ADMM algorithm.
